# Sample data for Tessera vignettes

Subset of MERFISH lung cancer dataset (Chen et al 2024).

## Usage

``` r
tessera_warmup
```

## Format

counts and meta data table for 3177 cells.

- counts:

  dgCMatrix of gene-by-cell counts

- meta_data:

  data.frame with spatial locations (XY) and cell lineage (type)
